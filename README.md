### 🔐 Authentication & Security

* Role-based login (Entrepreneur / Investor)
* Password validation
* 2FA (OTP mock)

---

### 📊 Dashboards

* Separate dashboards for Entrepreneurs & Investors
* Real-time stats (connections, meetings, requests)
* Role-based UI rendering

---

### 🤝 Collaboration System

* Send & receive collaboration requests
* Accept / Reject functionality
* Dynamic updates on dashboard

---

### 📅 Meeting Scheduler

* Schedule meetings via calendar
* Accepted meetings appear on dashboard
* Option to remove meetings

---

### 📹 Video Call (UI)

* Dedicated video call interface
* Ready for real-time integration

---

### 📂 Document Chamber

* Upload & preview documents
* E-signature (mock signature pad)
* Status tracking:

  * Draft
  * In Review
  * Signed
* Remove uploaded files

---

### 💳 Payment System (Mock)

* Wallet balance display
* Deposit / Withdraw / Transfer simulation
* Transaction history
* Investor → Entrepreneur funding flow

---

### 📱 Responsive Design

* Fully responsive across devices
* Mobile-friendly UI with adaptive layout
